# Soil organic carbon (SOC) sequestration potential and optimal policies

Software to accompany the paper at https://arxiv.org/abs/2409.18198

- 'functions.R' includes the main functions that are used for calculating optimal policies
- 'simulations.R' includes some additional functions to construct and run the simulations in the paper
- 'sequestration_potential.Rmd' includes data analysis, simulation implementation, and results
